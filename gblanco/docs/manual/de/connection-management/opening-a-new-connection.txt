Eine neue Datenbankverbindung mit Doctrine herzustellen ist sehr einfach.
Wenn Sie PDO (www.php.net/PDO) benutzen wollen, erzeugen Sie einfach ein neues PDO Objekt:

<code type="php">
$dsn = 'mysql:dbname=testdb;host=127.0.0.1';
$user = 'dbuser';
$password = 'dbpass';

try {
    $dbh = new PDO($dsn, $user, $password);
    $conn = Doctrine_Manager::connection($dbh);
} catch (PDOException $e) {
    echo 'Connection failed: ' . $e->getMessage();
}
</code>

Anmerkung: Wenn Sie eine PDO Instanz direkt an Doctrine_Manager::connection() übergeben, kennt Doctrine den Usernamen/Passwort der Verbindung nicht, da sich diese Informationen nicht aus einer bestehenden PDO Instanz auslesen lassen. Username und Passwort werden von Doctrine aber benötigt, um Datenbanken zu erzeugen und zu löschen.
Dieses Problem können Sie umgehen, indem sie dem $conn Objekt manuell Username und Passwort zuweisen.

<code type="php">
$conn->setOption('username', 'username');
$conn->setOption('password', 'password');
</code>
